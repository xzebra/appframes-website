## What is AppFrames?

AppFrames is a powerful Mac application designed to help developers and designers create professional app screenshots and mockups. Whether you're preparing assets for the Mac App Store or creating marketing materials, AppFrames provides the tools you need to showcase your app beautifully.

### Key Features

- **Device Frames**: Add realistic device frames to your screenshots
- **Multiple Templates**: Choose from various layout templates
- **Layer Management**: Organize your content with powerful layer tools
- **Export Options**: Export in various formats and resolutions
- **Dark Mode Support**: Create screenshots for both light and dark themes
